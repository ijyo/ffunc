# ffunc

zsh utilities with [fzf](https://github.com/junegunn/fzf).

## Installation

### e.g. [zinit](https://github.com/zdharma/zinit)

```zsh
zinit ice src'init.zsh'
zinit light ijyo/ffunc.git
```

Default settings is see the `init.zsh`.
